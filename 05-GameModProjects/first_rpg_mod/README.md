# 游戏扩展与模块化改造项目
## 概述
本项目是我对一款基于RPG Maker引擎的游戏进行的功能扩展实践。核心是：**在不修改游戏核心引擎的前提下，通过逆向工程与模块化设计，为游戏的一个角色新增入队功能及其配套交互系统**。
## 项目亮点
- **系统性**：完整经历了逆向分析、系统解构、模块设计、兼容调试等全流程；
- **技术深度**：深入理解了游戏的状态管理、对话系统结构等，并解决了其中的兼容性问题；
- **工程思维**：实践了模块解耦等软件工程思想，确保功能的可维护性。
